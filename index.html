Setup secure authentication in under 5 minutes

Providing central user authentication typically is a burden, requiring a network admin battling active directory for days, if not weeks; you need to invest in beefy servers and perform regular maintenance.
→ With GLAuth, security is as easy as running a single, portable executable on the OS of your choice, and ocasionally updating your users' list.

Use Cases

Progressive Rollout

Keep using your legacy systems. GLAuth, as a gateway, injects security features like Two-Factor or TLS support.

Complete Deployment

Use GLAuth standalone and implement support for groups, SSH keys, home environments, Two-Factor, app passwords and physical tokens.

Backends

Store your users in your backend of choice: simple flat files, local or in S3; OwnCloud; SQL Databases or LDAP.
